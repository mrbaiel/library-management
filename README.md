# Library Management Backend

REST API для управления библиотекой: книги, авторы, избранное, периодические задачи.

## Стек

- Python 3.12+ / Django 5 / DRF
- PostgreSQL
- JWT (simplejwt)
- Celery + Redis
- Swagger (drf-spectacular)
- Docker
